# 변경 이력 (CHANGELOG)

## [v0.2.0] - 2026-03-26

### 🎮 새로운 기능
- **플랫폼 통과 기능**: 아래 방향키를 누른 상태에서 점프하면 발판을 통과해서 아래층으로 떨어질 수 있습니다
  - Celeste, Hollow Knight 스타일
  - 바닥(맨 아래층)은 통과 불가능
  - 0.33초 타이머로 실수 방지

### 🔊 사운드 시스템
Web Audio API를 사용한 8비트 레트로 사운드 완전 구현:
- **점프 사운드**: 상승 톤 (Square wave)
- **코인 수집**: 경쾌한 팅! 소리 (Sine wave)
- **콤보**: 더 높은 톤의 피드백 (Triangle wave)
- **아이템 획득**: 반짝이는 듀얼 톤
- **대시**: 화이트 노이즈 기반 쉿! 소리
- **피격**: 낮은 톤 경고음 (Sawtooth wave)
- **스테이지 클리어**: C-E-G-C 팡파레
- **게임 오버**: 하강 톤 (Sawtooth wave)

### 🎯 개선사항
- 모바일 터치에서도 플랫폼 통과 가능 (D패드 아래 + JUMP)
- 오디오 컨텍스트 자동 활성화 (모바일 대응)
