## ✨ 새로운 기능

### 1. 플랫폼 통과 기능 (Drop-Through Platforms)
**아래 키 + 점프**로 발판을 통과해서 아래층으로 떨어질 수 있습니다!

#### 특징
- ⬇️ + Space (PC) 또는 D패드 아래 + JUMP (모바일)
- Celeste, Hollow Knight 스타일의 플랫폼 통과
- 바닥(맨 아래층)은 통과 불가능 (안전장치)
- 0.33초 타이머로 실수로 인한 통과 방지
- 모든 중간층에서 작동

#### 코드 구현
```javascript
// 플레이어 상태에 fallThrough 타이머 추가
fallThrough: 0

// 점프 시 아래 키 확인
if (this.keys.down) {
  p.fallThrough = 20; // 20프레임 활성화
}

// 플랫폼 충돌에서 통과 로직
if (this.keys.down && p.fallThrough > 0 && pl.y !== this.platforms[0].y)
  continue; // 플랫폼 통과
```

### 2. 8비트 레트로 사운드 시스템
Web Audio API로 구현한 완전한 사운드 시스템!

#### 사운드 목록

| 이벤트 | 사운드 타입 | 주파수 | 설명 |
|--------|------------|--------|------|
| 점프 | Square Wave | 300→500Hz | 상승 톤 |
| 코인 수집 | Sine Wave | 800→1200Hz | 경쾌한 팅! |
| 콤보 | Triangle Wave | 1000→1500Hz | 높은 톤 피드백 |
| 아이템 | Dual Sine | 600+900Hz | 반짝이는 화음 |
| 대시 | White Noise | 2000Hz+ | 쉿! 소리 |
| 피격 | Sawtooth | 200→50Hz | 낮은 경고음 |
| 클리어 | Triangle | C-E-G-C | 팡파레 멜로디 |
| 게임오버 | Sawtooth | 400→100Hz | 하강 톤 |

#### 특징
- 100% 절차적 생성 (외부 파일 불필요)
- 모바일 호환 (터치 시 자동 활성화)
- 레트로 아케이드 느낌
- 평균 0.1~0.3초 재생 시간

### 3. 파티클 효과 시스템
모든 이벤트에 화려한 파티클 효과 추가!

#### 파티클 종류

| 이벤트 | 색상 | 개수 | 설명 |
|--------|------|------|------|
| 과일 수집 | 🔵 Cyan | 8개 | 방사형 버스트 |
| 콤보 | 🟡 Yellow | 8개 | 더 밝은 버스트 |
| 아이템 | 🟣 Purple | 12개 | 화려한 폭발 |
| 대시 | 💙 Light Blue | 3개/프레임 | 트레일 효과 |

#### 물리 시뮬레이션
```javascript
// 각 파티클마다
pt.vy += 0.2;    // 중력
pt.vx *= 0.98;   // 마찰/공기저항
pt.life--;       // 수명 감소

// 페이드 아웃
alpha = Math.min(1, pt.life / 15);
```

#### 특징
- 30-45 프레임 수명
- 알파 페이드 아웃
- 방사형 버스트 패턴
- 중력 + 마찰 물리
