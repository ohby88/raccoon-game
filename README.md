# RACCOON ADVENTURE

레트로 아케이드 스타일 플랫포머 게임. HTML5 Canvas + 순수 JavaScript로 구현.
토스 미니앱 탑재를 목표로 개발 중.

## 플레이 방법

`index.html` 파일을 브라우저에서 열면 바로 실행됩니다. 별도 설치 불필요.

### 조작키
| 동작 | PC | 모바일 |
|------|-----|--------|
| 이동 | 방향키 / WASD | D패드 |
| 점프 | Space / W / ↑ | JUMP 버튼 |
| **아래층 떨어지기** | **↓ + Space** | **D패드 아래 + JUMP** |
| 사다리 오르기 | ↑ / W | D패드 위 |
| 사다리 내려가기 | ↓ / S | D패드 아래 |
| 사다리 이탈 | Space | JUMP 버튼 |
| 토끼 대시 | Space 두 번 | JUMP 두 번 |

## 캐릭터

| 캐릭터 | 해제 조건 | 특기 |
|--------|----------|------|
| 너구리 | 처음부터 | 밸런스형 |
| 여우 | 스테이지 3 클리어 | 2단 점프 |
| 토끼 | 스테이지 7 클리어 | 대시 |

## 게임 시스템

- **20스테이지** — 스테이지 1~5 수작업, 6~20 절차적 생성
- **콤보 시스템** — 연속 과일 수집 시 x2~x10 점수 배율 (3초 내 다음 과일)
- **TIME BONUS** — 빠른 클리어일수록 추가 점수
- **스테이지 아이템** — ★무적 / ❄시간정지 / ⊕자석 / ⚡속도부스트
- **미션 8종** — 과일 수집, 스테이지 클리어, 콤보 달성 등
- **데일리 챌린지** — 매일 다른 조건 (스피드런 / 노아이템 / 풀콤보 / 목숨유지)
- **TOP SCORE** — 로컬 저장

## 파일 구조

```
raccoon-game/
├── index.html   # 게임 UI, 캐릭터 선택, 터치 컨트롤
└── game.js      # 게임 엔진 전체 (물리, 렌더, AI, 스테이지, 시스템)
```
